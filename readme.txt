Get instant visual and audio notifications when new paid WooCommerce orders arrive.

== Description ==

Best WooCommerce Alerts provides real-time notifications for your WooCommerce store, ensuring you never miss a paid order. When a new order is received and paid for, the plugin triggers both visual and audio alerts to grab your attention immediately.

= Key Features =

* **Real-Time Alerts**: Instant notifications when new paid orders are received
* **Visual Alerts**: Full-screen red flash notification for maximum visibility
* **Audio Alerts**: Customisable sound notifications
* **Smart Filtering**: Only alerts for paid orders (processing or completed status)
* **Connection Status**: Visual indicator showing real-time connection status
* **Mobile Friendly**: Works perfectly on all devices
* **Easy Setup**: No configuration needed - works out of the box
* **Customisable**: Change alert sounds via WordPress media library or custom URL

= Perfect For =

* Busy store owners who need to process orders quickly
* Restaurants and food delivery services
* Any business requiring immediate order notification
* Stores with time-sensitive products or services

= Requirements =

* WordPress 5.0 or higher
* WooCommerce 6.0 or higher
* Modern web browser with JavaScript enabled

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/best-woocommerce-alerts` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to WooCommerce > Order Alerts to customise your alert sound (optional)

== Frequently Asked Questions ==

= Does this work with all WooCommerce orders? =

No, by design it only alerts for paid orders (status: processing or completed) to avoid false alerts for pending or failed orders.

= Can I customise the alert sound? =

Yes, you can upload your own sound file through the WordPress media library or enter a URL to any MP3 file.

= Will this work on mobile devices? =

Yes, the plugin is fully responsive and works on all devices with a modern web browser.

= Does this slow down my site? =

No, the plugin is lightweight and only runs in the WordPress admin area.
